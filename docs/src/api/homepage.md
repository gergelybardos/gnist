# Gnist API Reference

Welcome to the technical specification of the Gnist API.

---

## Important Information

- **Versioning:** This reference reflects the **current stable version** of Gnist. Documentation of past versions is not kept online. Version-specific history is maintained via Git tags and the project `CHANGELOG.md`.
- **Scope:** This reference covers only the **public-facing API** of Gnist. Private internals are intentionally omitted.
